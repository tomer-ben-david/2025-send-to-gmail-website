Frequently Asked Questions

Is my data secure?

Yes! Send to Gmail uses Google's official OAuth 2.0 authentication. Your credentials are never stored locally, and all communication is encrypted.

Can I use it with multiple Google accounts?

Currently, the app supports one Google account at a time. You can switch accounts by signing out and back in.

What file types are supported?

All file types are supported! The app works with any file that Gmail can handle as an attachment.

Is there a cost?

Send to Gmail is completely free to use. No subscriptions, no hidden fees.
